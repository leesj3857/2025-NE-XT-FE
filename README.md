본 프로젝트는 React 기반으로 제작되었으며, 개발을 시작하기 전에 아래 내용을 참고해주세요.

---

## ⚙️ 설치 및 실행 방법

1. .env 파일 생성

   VITE_KAKAO_REST_API_KEY= 발급받은 REST_API_KEY
   VITE_KAKAO_JS_KEY= 발급받은 JAVASCRIPT_API_KEY
    


```bash
# 1. 필요한 패키지 설치
npm install

# 2. 개발 서버 실행
npm run dev
```
